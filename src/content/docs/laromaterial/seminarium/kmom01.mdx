1. Gör ett program som hanterar bedömning av olika sorters glassar.I detta fall är det maximala antalet olika glassar som kan bedömas 10.
Metoder ska definieras och användas för att hantera olika delproblem.

Exempelkörning:  

Användarinmatning visas med kursiv text.  

Hur många glassar ska bedömas (max 10)? 6  

Mata in namnen på de 6 glassarna:  
<i>Magnum mandel</i>
*Storstrut*  
*Piggelin*  
*88:an*  
*Nogger*  
*Cornetto jordgubb*  


Ge bedömning (1-5):  
Magnum mandel : *5*  
Storstrut : *2*  
Piggelin : *2*  
88:an: *5*  
Nogger : *4*  
Cornetto jordgubb: *3*  
  
Fler bedömningar (J/N)? *J*  
  
Ge bedömning (1-5):  
Magnum mandel : *4*  
Storstrut : *1*  
Piggelin : *4*  
88:an: *5*  
Nogger : *3*  
Cornetto jordgubb: *2*  
  
Fler bedömningar (J/N)? *N*  
  
Antalet genomförda bedömningar är 2  
Medelvärdet för varje glass blev  

Magnum mandel : 4.5  
Storstrut : 1.5  
Piggelin : 3  
88:an: 5  
Nogger : 3.5  
Cornetto jordgubb: 2.5  

Glassen som fick högst poäng är 88:an.  
Glassen som fick lägst poäng är Storstrut.

- En konstant ska användas för det maximala antalet glassar (10)
